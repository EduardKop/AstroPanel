# PWA Setup Instructions

## Icons Missing
The following icon files need to be created and placed in `/public/`:

1. **icon-192.png** - 192x192px app icon
2. **icon-512.png** - 512x512px app icon

You can use any online tool like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Or design custom icons in Figma/Photoshop

### Suggested Design:
- Dark background (#0A0A0A)
- Blue/purple gradient accents
- Modern, minimalist look
- Clear even at small sizes

## Testing PWA Installation

### On iPhone:
1. Open the app in Safari
2. Tap the Share button (square with arrow)
3. Scroll and tap "Add to Home Screen"
4. The app will install with the icon

### On Android:
1. Open the app in Chrome
2. Tap the menu (three dots)
3. Tap "Install app" or "Add to Home Screen"
4. The app will install with the icon

### On Desktop (Chrome/Edge):
1. Look for the install icon in the address bar
2. Click "Install AstroPanel"
3. App opens in standalone window
